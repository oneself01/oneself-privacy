<h1>プライバシーポリシー</h1>

<p>ONESELF（以下、「当社」といいます。）は、当社が提供する「ONESELF Attendance」（以下、「本アプリ」といいます。）における利用者の個人情報の取り扱いについて、以下のとおりプライバシーポリシーを定めます。</p>

<h2>1. 取得する情報</h2>
<p>本アプリでは、以下の情報を取得します。</p>
<ul>
  <li>氏名</li>
  <li>メールアドレス</li>
  <li>打刻情報（出勤・退勤・休憩開始・休憩終了）</li>
  <li>位置情報（GPS）※出勤・退勤の位置情報確認のため</li>
  <li>アプリ利用ログ（操作履歴・エラー情報）</li>
</ul>

<h2>2. 情報の利用目的</h2>
<p>取得した情報は以下の目的で使用します。</p>
<ul>
  <li>勤怠管理・出退勤状況の記録</li>
  <li>有給管理・休日管理</li>
  <li>社内業務改善のための分析</li>
  <li>不正打刻の防止のための位置情報確認</li>
  <li>アプリの改善および不具合対応のため</li>
</ul>

<h2>3. 情報の第三者提供</h2>
<p>利用者の同意なく第三者へ提供することはありません。ただし、法令に基づく場合を除きます。</p>

<h2>4. 外部サービスの利用</h2>
<p>本アプリでは、以下の外部サービスを利用する場合があります。</p>
<ul>
  <li>Google Apps Script（バックエンド処理）</li>
  <li>Google Sheets（勤怠データ保存）</li>
  <li>Firebase Hosting / Cloudflare（バックエンドAPI）</li>
</ul>

<h2>5. 位置情報の扱いについて</h2>
<p>位置情報は、出勤・退勤時の不正防止目的でのみ使用します。常時取得はしません。</p>
